## Sobre este projeto

O Radar Imobiliário do Vale do Aço é uma publicação de análise de dados aplicada ao mercado imobiliário de Ipatinga, Coronel Fabriciano, Timóteo e região metropolitana. Todas as análises são baseadas em fontes primárias verificáveis e metodologia equivalente à adotada pelo Índice FipeZAP.

Este repositório é mantido por **Wederson Marinho** — Data Scientist, CRECI-MG 58.263-F, CNAI 51.239.

## Fontes de dados utilizadas

Todos os dados são públicos e verificáveis:

- **IBGE Cidades** — População estimada, PIB per capita
- **FipeZAP / DataZap** — Índice de preços imobiliários (acumulados anuais)
- **ANOREG/BR** — Análise FipeZAP 2025
- **Banco Central do Brasil** — Taxa Selic
- **IBGE / IPCA** — Inflação acumulada 12 meses
- **Caravela.info** — Emprego formal e empresas abertas
- **Zap Imóveis / VivaReal / OLX** — Anúncios ativos de venda e locação
- **Imobiliárias locais** — JL Imóveis, WF Imóveis, Porta Aberta, JF Corretor, Ciclo Imóveis

---

## Metodologia

O preço por m² é calculado sobre a área privativa declarada em anúncios públicos nos portais Zap Imóveis, VivaReal e OLX — metodologia equivalente à adotada pelo Índice FipeZAP para cidades monitoradas. Não representa preço de transação efetivada.
